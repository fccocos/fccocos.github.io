# 数列极限的定义

ε−N语言：lim⁡n→∞xn=a⇔∀ε>0,∃N∈N+,当n>N时，恒有∣xn−a∣<ϵ\varepsilon-N 语言 ： \lim\limits_{n\to \infty} x_n = a \Leftrightarrow \forall \varepsilon > 0, \exist N \in N_+, 当n>N时，恒有|x_n - a| < \epsilonε−N语言：n→∞lim​xn​=a⇔∀ε>0,∃N∈N+​,当n>N时，恒有∣xn​−a∣<ϵ

极限的定义证明的步骤

1️⃣先写距离 ∣xn−a∣<ε|x_n - a| < \varepsilon∣xn​−a∣<ε

2️⃣反解出 n 的取值范围: n>g(ε)n > g(\varepsilon)n>g(ε)

3️⃣取N=⌈g(ε)⌉=[g(ε)]+1N = \lceil g(\varepsilon)\rceil = [g(\varepsilon)]+1N=⌈g(ε)⌉=[g(ε)]+1

定理 若数列an{a_n}an​ 收敛，则其任意子列ank{a_{n_k}}ank​​ 也收敛，且lim⁡k→∞ank=lim⁡n→∞an\lim\limits_{k\to \infty}a_{n_k} = \lim\limits_{n \to \infty}a_nk→∞lim​ank​​=n→∞lim​an​

上述定理的逆否命题也成立

对于数列{an}\{a_n\}{an​}, 如果其存在一个子列{ank}\{a_{n_k}\}{ank​​} 发散，或lim⁡k→∞ank≠lim⁡n→∞an\lim\limits_{k\to \infty}a_{n_k} \ne \lim\limits_{n \to \infty}a_nk→∞lim​ank​​=n→∞lim​an​，则数列{an}\{a_n\}{an​} 发散

lim⁡n→∞an=a⇔lim⁡k→∞a2k=a,且lim⁡k→∞a2k−1=a\lim\limits_{n\to \infty}a_n=a \Leftrightarrow \lim\limits_{k\to \infty} a_{2k} = a, 且\lim\limits_{k\to \infty} a_{2k-1} = an→∞lim​an​=a⇔k→∞lim​a2k​=a,且k→∞lim​a2k−1​=a

# 定理 3 (保号性)

（脱帽法）如果极限值 a 存在且大于 0 (小于 0)，则存在正整数 N>0, 使得当 n>N 时， 有an>0a_n>0an​>0(an<0a_n<0an​<0)

（戴帽法）推论：数列{an}\{a_n\}{an​} 从某项开始其有an≥0,且lim⁡n→∞an=a,则a≥0a_n\ge0,且\lim\limits_{n\to \infty}a_n=a, 则a\ge0an​≥0,且n→∞lim​an​=a,则a≥0

# 极限运算规则

设lim⁡n→∞xn=a,lim⁡n→∞yn=b,则设\lim\limits_{n\to \infty}x_n =a, \lim\limits_{n\to \infty}y_n = b,则设n→∞lim​xn​=a,n→∞lim​yn​=b,则

  1. lim⁡n→∞(xn±yn)=a±b\lim\limits_{n\to \infty}(x_n\pm y_n)=a\pm bn→∞lim​(xn​±yn​)=a±b
  2. lim⁡n→∞xnyn=ab\lim\limits_{n\to \infty}x_ny_n=abn→∞lim​xn​yn​=ab
  3. 若a\ne0,y_n\ne0, 则\lim\limits_{n\to \infty}\frac{x_n}{y_n}=\frac{a}

# 夹逼定理

如果数列{xn},{yn},{zn}\{x_n\}, \{y_n\},\{z_n\}{xn​},{yn​},{zn​} 满足一下条件

{(1)yn≤xn≤zn(2)lim⁡n→∞yn=a,lim⁡n→∞zn=a\begin{cases} (1)\ \ \ y_n\le x_n \le z_n \newline (2)\ \ \ \lim\limits_{n \to \infty}y_n = a , \lim\limits_{n \to \infty}z_n = a \end{cases}{(1)  yn​≤xn​≤zn​(2)  n→∞lim​yn​=a,n→∞lim​zn​=a​

则数列{xn}\{x_n\}{xn​} 的极限存在，且lim⁡n→∞xn=a\lim\limits_{n\to \infty}x_n = an→∞lim​xn​=a

# 单调有界准则

单调有界数列必有极限，即若数列{xn}\{x_n\}{xn​} 单调增加 (减少) 且有上界 (下界)，则lim⁡n→∞xn\lim\limits_{n\to \infty}x_nn→∞lim​xn​ 存在
