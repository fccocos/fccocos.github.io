# 1. 常数项级数

# 一、 级数的概念和性质

# 1. 定义

有一个数列{un},则称∑n=1∞un=u1+u2+...+un+...为无穷级数，则{sn}为级数∑n=1∞un的部分和数列,如果部分和数列sn有极限s,即lim⁡x→∞sn=s则称级数∑n=1∞un收敛，,这时极限s叫做级数∑n=1∞un的和。如果{sn}没有极限，此时称级数∑n=1∞un发散。若级数∑n=1∞un收敛于s,此时称rn=s−sn=un+1+un+2+...为级数的余部。显然，如果级数∑n=1∞un收敛,则lim⁡n→∞rn=0。\begin{aligned} &有一个数列\{u_n\},则称\sum_{n=1}^\infty u_n=u_1+u_2+...+u_n+...为无穷级数，\\ &则\{s_n\}为级数\sum_{n=1}^\infty u_n的部分和数列,如果部分和数列s_n有极限s,即\\ &\lim_{x\to\infty}s_n=s\\ &则称级数\sum_{n=1}^\infty u_n收敛，,这时极限s叫做级数\sum_{n=1}^\infty u_n的和。\\ &如果\{s_n\}没有极限，此时称级数\sum_{n=1}^\infty u_n发散。\\ &若级数\sum_{n=1}^\infty u_n收敛于s,此时称r_n=s-s_n=u_{n+1}+u_{n+2}+...为级数的余部。\\ &显然，如果级数\sum_{n=1}^\infty u_n收敛,则\lim_{n\to\infty}r_n=0。 \end{aligned}​有一个数列{un​},则称n=1∑∞​un​=u1​+u2​+...+un​+...为无穷级数，则{sn​}为级数n=1∑∞​un​的部分和数列,如果部分和数列sn​有极限s,即x→∞lim​sn​=s则称级数n=1∑∞​un​收敛，,这时极限s叫做级数n=1∑∞​un​的和。如果{sn​}没有极限，此时称级数n=1∑∞​un​发散。若级数n=1∑∞​un​收敛于s,此时称rn​=s−sn​=un+1​+un+2​+...为级数的余部。显然，如果级数n=1∑∞​un​收敛,则n→∞lim​rn​=0。​

# 2. 性质

(1)∑n=1∞un和∑n=1∞kun同敛散，其中k为任意不为零的常数。(2)∑n=1∞un和∑n=1∞vn分别收敛于S、δ,则∑n=1∞(un+vn)=S+δ.(3)改变级数前有限项不改变级数的敛散性.(4)收敛级数加括号后敛散性不变，且和不变(5)级数收敛的必要条件是lim⁡n→∞un=0\begin{aligned} &(1) \sum_{n=1}^\infty u_n和\sum_{n=1}^\infty ku_n同敛散，其中k为任意不为零的常数。\\ &(2) \sum_{n=1}^\infty u_n和 \sum_{n=1}^\infty v_n分别收敛于S、\delta,则\sum_{n=1}^\infty (u_n+v_n)=S+\delta.\\ &(3)改变级数前有限项不改变级数的敛散性.\\ &(4)收敛级数加括号后敛散性不变，且和不变\\ &(5) 级数收敛的必要条件是\lim_{n\to\infty} u_n=0 \end{aligned}​(1)n=1∑∞​un​和n=1∑∞​kun​同敛散，其中k为任意不为零的常数。(2)n=1∑∞​un​和n=1∑∞​vn​分别收敛于S、δ,则n=1∑∞​(un​+vn​)=S+δ.(3)改变级数前有限项不改变级数的敛散性.(4)收敛级数加括号后敛散性不变，且和不变(5)级数收敛的必要条件是n→∞lim​un​=0​

# 二、 级数的审敛法

# 1. 正项级数

# 1） 比较判别法

前提条件：un≥0前提条件：u_n\ge0前提条件：un​≥0

如果0≤un≤vn,则∑n=1∞vn收敛⇒∑n=1∞un;∑n=1∞un发散⇒∑n=1∞vn发散;如果0\le u_n\le v_n,则\sum_{n=1}^\infty v_n收敛 \Rightarrow \sum_{n=1}^\infty u_n; \sum_{n=1}^\infty u_n发散 \Rightarrow \sum_{n=1}^\infty v_n 发散;如果0≤un​≤vn​,则n=1∑∞​vn​收敛⇒n=1∑∞​un​;n=1∑∞​un​发散⇒n=1∑∞​vn​发散;

# 2）比较判别法的极限形式

设lim⁡n→∞unvn=l(0≤l≤+∞)如果0<l<+∞,则∑n=1∞un和∑n=1∞vn同敛散如果l=0,∑n=1∞vn收敛⇒∑n=1∞un收敛如果l=+∞,∑n=1∞vn发散⇒∑n=1∞vn发散设\lim_{n\to\infty}\frac{u_n}{v_n}=l(0\le l\le +\infty)\\ 如果0< l < {+}\infty,则\sum_{n=1}^\infty u_n 和\sum_{n=1}^\infty v_n同敛散\\ 如果l=0,\sum_{n=1}^\infty v_n收敛 \Rightarrow \sum_{n=1}^\infty u_n收敛\\ 如果l=+\infty,\sum_{n=1}^\infty v_n发散 \Rightarrow \sum_{n=1}^\infty v_n发散设n→∞lim​vn​un​​=l(0≤l≤+∞)如果0<l<+∞,则n=1∑∞​un​和n=1∑∞​vn​同敛散如果l=0,n=1∑∞​vn​收敛⇒n=1∑∞​un​收敛如果l=+∞,n=1∑∞​vn​发散⇒n=1∑∞​vn​发散

# 3. 任意项级数

# 1）绝对收敛和条件收敛的条件

数列∑n=1∞∣un∣收敛，称为绝对收敛。数列∑n=1∞∣un∣发散，∑n=1∞un收敛，称为条件收敛。\begin{aligned} &数列\sum_{n=1}^\infty |u_n|收敛，称为绝对收敛。\\ &数列\sum_{n=1}^\infty |u_n|发散，\sum_{n=1}^\infty u_n收敛，称为条件收敛。 \end{aligned}​数列n=1∑∞​∣un​∣收敛，称为绝对收敛。数列n=1∑∞​∣un​∣发散，n=1∑∞​un​收敛，称为条件收敛。​

# 2）关于绝对收敛和条件收敛的基本结论

1.绝对收敛的级数必定收敛2.条件收敛的级数所有正项级数（负项级数）发散\begin{aligned} &1.绝对收敛的级数必定收敛\\ &2.条件收敛的级数所有正项级数（负项级数）发散 \end{aligned}​1.绝对收敛的级数必定收敛2.条件收敛的级数所有正项级数（负项级数）发散​
