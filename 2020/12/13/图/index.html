# 一、图的概念

# 1. 图的定义

图 G 由定点集合 V 和边集 E 组成，记作 G=(V,E),V (G) 表示图 G 中顶点的有限非空集合，E (G) 表示图 G 中顶点之间的关系集合（边）。

若 V={v1,v2,v3,...,vnv_1,v_2,v3,...,v_nv1​,v2​,v3,...,vn​}，则用 | V | 表示图 G 中顶点的个数，即图的阶数。

E={(u,v)|v 属于 V,u 属于 V}，用 | E | 表示图 G 边的条数。

线性表可以为空，树可以为空，但图一定不能为空。也就是说图中不能一个顶点都没有，图的顶点集一定非空，但图的边集可以为空

# 2. 有向图

若 E 是有向边的有限集合，则图 G 为有向图。

边是顶点的无序对，记为 <v, w>，其中 v,w 为顶点，v 称为有向边的尾，w 称为有向边的头.

<v,w> 称为从顶点 v 到顶点 w 的边，或称为顶点 v 邻接

# 二、图的存储结构及基本操作

# 1. 邻接矩阵存储结构的定义

#define MaxVertexNum 100
typedef char VertexType;
typedef int EdgeType;
typedef struct{
    VertexType Ver[MaxVertexNum];
    EdgeType Edge[MaxVertexNum][MaxVertex];
    int vexnum,arcnum;

# 邻接矩阵存储表示法具有以下特点:

  1. 无向图的邻接矩阵是一个对称矩阵。因此，实际存储时只需存储上（或下）三角矩阵的元素。
  2. 对于有向图，邻接矩阵的第 i 行非零元素的个数正好是第 i 个顶点的出度或入度。邻接矩阵的第 j 列非零元素的个数正好是第 j 个顶点的出度或入度。
  3. 稠密图适合用邻接矩阵的存储表示。
  4. An的元素表示An[i][j]等于由顶点i到顶点j的长度为n的路径的数目A^n的元素表示A^n[i][j]等于由顶点i到顶点j的长度为n的路径的数目An的元素表示An[i][j]等于由顶点i到顶点j的长度为n的路径的数目
  5. 用邻接矩阵存储表示法，很容易确定图中任意两个顶点之间是否连通。但，要确定图中的边数，则必须按行、列对每一个元素进行检查，所花费的时间代价大。

# 三、 图的遍历

# 1. 图的广度优先搜索

广度优先搜索的伪代码

bool visited[MAX_VEXTEX_NUM];
void BFSTraverse(Graph G)
{
    for(int i=0;i<G.VexNum;i++)
    	visited[i]=false;
    InitQueue(Q);
    for(int i=0;i<G.VexNum;i++)
        if(!visited[i])
           BFS(G,i);
}

void BFS(Graph G,int v)
{
    visit(v);
    visited[v]=true;
    EnQueue(Q,v);
    while(!isEmpty(Q))
    {
        DeQueue(Q,v);
        for(int w=FirstNeighbor(G,v);w>=0;w=NextNeighbor(G,v.w))
            if(!visited[w])
            {
                visit(w);
                visited[w]=false;
                EnQueue(w);
            }
    }
}

BFS 的空间复杂度为 O (|V|)

BFS 的时间复杂度:

​ 当为邻接矩阵存储表示法时，O (∣v∣2|v|^2∣v∣2)

​ 当为邻接表存储表示法时，O (|v|+|E|)

BFS 算法解决单源最短路径问题的算法

#define infity 65535
void BFS_Min_Distance(Graph G,int u)
{
    int d[Max_Vertex_Num];
    for(int i=0;i<G.VexNum;i++)
        d[i]=infity;
    visited[u]=true;
    d[u]=0;
    InitQueue(Q);
    EnQueue(Q,u);
    while(!isEmpty(Q))
    {
        DeQueue(Q,u);
        for(int w=FirstNeighbor(G,u);w>=0;w=NextNeighbor(G,u,w))
            if(!visited[w])
            {
                visited[w]=true;
                d[w]=d[u]+1;
                EnQueue(Q,w);
            }
    }
}

# 2. 图的深度优先遍历

深度优先遍历伪代码

bool visited[MAX_VERTEX_NUM];
void DFSTraverse(Graph G)
{
	for(int i=0;i<G.VexNum;i++)
        visited[i]=false;
    for(int i=0;i<G.VexNum;i++)
        if(!visited[i])
            DFS(G,i);
}
void DFS(Graph G,int v)
{
    visit(v);
    visited[v]=true;
    for(int w=FirstNeighbor(G,w);w>=0;w=NextNeighbor(G,v,w))
    {
        if(!visited[w])
            DFS(G,w);
    }
    
}

空间复杂度为 O (|V|)

时间复杂度为 O (∣V∣2|V|^2∣V∣2)(邻接矩阵)，O (|V|+|E|)(邻接表)

# 四、图的应用

# 1. 最小生成树

# prim 算法

void prim(Graph G,Graph &T)
{
//说明：G={V,E},V为顶点集，E为边集；T={U,E},U为最小生成树的顶点集，E为最小生成树中的边集
//1.初始化：向T中添加G中的任意一个顶点u,使得U={u}，E为空
 T=空;
 U={w};
//2.循环（结束条件为U==V）：从G中选出满足{(u,v)|u属于U,v属于V-U}且具有最小权值的边(u,v),加入到树T中，置U=UU{u},E=EU{(u,v)}
 while((V-U)!=空)
 {
     设(u,v)使u属于U,v属于V-U,且权值最小的边;
     T=T U {(u,v)};
     U=U U {v};
 } 

}

prim 算法的时间复杂度为 O (∣V∣2|V|^2∣V∣2)

# Kruskal 算法

设G=(V,E);T=(U,Et);
初始化：U=V,Et=空
循环（结束条件为生成一棵树）：按G的边的权值递增顺序依次从E-Et中选择一条边，若这条边加入到了T后不构成回路，则将该边加入到Et中，否则舍弃，直到Et含有n-1条边

void kruskal(G T)
{
	U=V;
    Et=空;
    numS=n;//连通分量数即顶点个数
    while(numS>1)
    {
        从E中选取权值最小的边(u,v);
        if(v和u属于T中不同的连通分量)
        {
            T=T U {(u,v)};
            numS--;
        }
    }
       
}

kruskal 算法中，采用堆来存放边的集合，每次选择最小权值的边的时间为 O (log|E|); 构造一棵最小生成树的时间复杂度为 O (|E|log|E|);kruskal 算法使用于边稀疏顶点稠密的图。

# 五、拓扑排序

拓扑排序：在图论中，有一个有向无环图的顶点组成的序列，当且仅当满足下列条件时，称为该图的一个拓扑排序：

  1. 每个顶点只出现一次
  2. 若顶点 A 在序列中排在顶点 B 的前面，则在图中不存在从顶点 B 到顶点 A 的路径。

即，拓扑排序是对有向无环图的顶点的一种排序，它使得若存在一条从顶点 A 到顶点 B 的路径，则顶点 B 一定出现在顶点 A 的后面。

每个 AOV 网都有一个或多个拓扑排序序列。

拓扑排序的实现步骤：

  1. 从 AOV 网中选取一个没有前驱的顶点并输出。
  2. 从网中删除该顶点和所有以他为起点的有向边
  3. 重复 1、2 步骤知道当前的 AOV 网为空或当前网中不存在无前驱的顶点为止。后一种情况说明有向图存在环。

拓扑排序的算法实现如下：

bool TopologicalSort(Graph G)
{
    InitStack(S);
    for(int i=0;i<G.VexNum;i++)
        if(indegree[i]==0)
            Push(S,i);
    int count=0;//记录当前已经输出的定点数
    while(!isEmpty(S))
    {
        Pop(S,i);
        print(count++)=i;//输出顶点i
        for(p=G.vertices[i].firstarc;p;p=p->nextarc)
        {
            v=p->adjvex;
            if(!(--indegree[v]))
            {
                push(S,v);
            }
        }
    }
    if(count<G.vexnum)
        return false;//排序失败，有向图中存在环
    else return true;//拓扑排序成功
    
}

int NextNeighbor(ALGraph &G,int x,int y)
{
    if(x!=-1)
    {
        ArcNode *p=G.vertices[x]->firstarc;
        while(p&&p->data!=y)
            p=p->nextarc;
        if(p&&p->data)
        {
            return p->next->data;
        }
    }
    return -1;
}
