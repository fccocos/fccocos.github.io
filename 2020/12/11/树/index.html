树和二叉树

树的基本概念

树的定义

树是n（n>=0）的有限集合。当n=0时，称为空树。

在任何一个非空树中应满足：

1）有且只有一个特定的称为根的节点。

2）当n>1时，其余剩余的m(m>0)个互不相交的有限集T1,T2,…,Tm,其中每个集合本身又是一个数，并称为根的子树。

树是一种递归的数据结构。

树作为一种逻辑结构，同时也是一种分层结构，具有以下两个特点：

  1. 树的根节点没有前驱，数根结点以外的所有结点都有且只有一个前驱节点
  2. 树中所有的节点可以有零个或多个后继

基本术语

树的度

在树中结点最大的度数。

节点的度

是指树中结点的孩子数。

分支节点（非终端节点）

是指度大于0的结点，即有孩子结点的结点。

叶子结点(终端结点)

是指度为0的结点，即没有孩子结点的结点。

节点的深度、高度、层次

深度：从根结点出发自顶向下逐层累加的

高度：从叶子节点出发自底向上逐层累加的

层次：从根节点开始定义的，根节点为第一层，其子树为第二层，以此类推。

无序树

树中结点的各个子树无序的，能够互相替换

有序树

树中结点的各个子树从左到右是有序的，不能相互替换。

路径和路径长度

路径：是指两个结点间，所经过的结点序列构成的

路径长度：是指路径上所经过的边的个数。

同一父结点的孩子结点之间是没有路径的。

森林

m(m>0)棵互不相交的树 的集合

树的基本性质

  1. 结点数=所有结点的度数之和+1
  2. 度为m的树在第i层的结点数至多有$m^{i-1}$个结点
  3. 高度为h的m叉树至多有$(m^h-1)/(m-1)$个结点
  4. 具有n个结点的m叉树的最小高度为$\left\lceil log_m{(n(m-1)+1)}\right\rceil$

二叉树的概念

二叉树的定义和其主要特性

二叉树的定义

二叉树树的另一种特殊形式，其特点是每个结点的度数不得大于2，并且二叉树的子树有左右之分，其次序不能颠倒。

二叉树是n($n\ge 0$)个结点的集合：

1）或者为空二叉树，即n=0；

2）或者由一个根节点和两个互不相交的被称为根的左子树和右子树组成。左右子树又可以分别为一棵二叉树。

二叉树是一棵有序树，但一棵度为2的有序树不是一棵二叉树。

二叉树与度为2的有序树的区别：

  1. 度为2的有序树至少有三个结点，而二叉树可以为空。
  2. 度为2的有序树的孩子的左右次序是相对于另一个孩子而言的，若某个结点只有一个孩子，则这个孩子没有左右之分，二二叉树无论其孩子数是否为2，均需要确定其左右次序，即二叉树中左右孩子的次序是确定的。

几个特殊的二叉树

满二叉树：高度为h，且含有$2^{h}-1$个结点的二叉树
  1. 满二叉树中每一层都含有最多的结点数
  2. 满二叉树的叶子节点都集中在二叉树的最下层
  3. 除叶子节点外的每一个节点的度为2.
完全二叉树：高度为h、有n个结点的二叉树，当且仅当其每一个结点都有与高度为h的满二叉树中编号为1~n的结点一一对应。
  1. 如果$i\le\left\lfloor n/2 \right\rfloor$,则结点i为分支节点，否则为叶子节点
  2. 叶子节点只可能出现在数的最后两层，对于最大层次中的叶子节点都依次排在该层的最左边。
  3. 若度为1的结点，则只可能有一个，且该结点只有左孩子，没有右孩子。
  4. 按层编号后，一旦出现某个结点为叶子结点或只有左孩子，则该结点以后的结点为叶子节点。
  5. 如果n 为奇数，则分支节点有左右孩子；如果n为偶数，则编号为n/2的结点只有左孩子
二叉排序树：左子树中关键字均小于根节点的关键字；右子树中关键字均大于根节点的关键字；左子树和右子树又各为一个排序树。
平衡二叉树：树中任意结点的左右子树的深度之差不超过1.

二叉树性质

  1. $n_0=n_2+1$==>结点树等与结点度数之和加1.
  2. 非空二叉树第k层至多有$2^{k-1}$个结点
  3. 高度为h的二叉树至多有$2^h-1$个点。
  4. 结点i所在层次为$\left\lfloor log_2i\right\rfloor+1$或$\left\lceil log_2{(i+1)}\right\rceil$
  5. 具有n个结点的完全二叉树的高度为$\left\lfloor log_2n\right\rfloor+1$或$\left\lceil log_2{(n+1)}\right\rceil$

二叉树的存储结构

二叉树的顺序存储:是指用一组连续的存储单元依次自顶向下、自左向右存储完全二叉树的结点元素，即将完全二叉树中编号为i的结点存储到在一维数组下标为i-1的分量中。

完全二叉树和满二叉树都适合用顺序存储结构存储。

一个高为h的二叉树只包含h个结点，用顺序结构存储却需要$2^h-1$个存储单元。

二叉树的链式存储

含有n个结点的二叉树中，含有n+1个空链域。

二叉树的遍历

先序遍历

如果二叉树为空，则什么都不做；否则：

  1. 访问根节点
  2. 遍历左子树
  3. 遍历右子树
void preOrder(BiTree T)
{
    if(T!=NULL)
    {
        visit(T);
        preOrder(T->lchild);
        preOrder(T->rchild);
    }
}

非递归的实现

  1. 先访问根节点，然后依次将根结点入栈，如果左孩子不为空，一直向左走，直到左孩子为空（说明此时已经访问到树的左下角）

  2. 如果左孩子为空，出栈元素，将其转向出栈结点的右子树，栈顶元素出栈

    void preOrder2(BiTree T)
{
   InitStack(S);
    BiTree p=T;
    while(T||!isEmpty(S))
    {
        if(p)
        {
            vist(p);
            Push(S,p);
            p=p->lchild;
        }else{
            pop(S,p);
            p=p->rchild;
        }

    }
}

中序遍历

如果二叉树为空，则什么都不做；否则：

  1. 遍历左子树
  2. 访问根节点
  3. 遍历右子树
void inOrder(BiTree T)
{
    if(T!=NULL)
    {
        inOrder(T->lchild);
        visit(T);
        inOrder(T->rchild)
    }
}

非递归的实现

  1. 沿着根的左孩子，依次入栈，直到左孩子为空（说明已经找到可以输出的结点）
  2. 栈顶元素出栈并访问元素：如果其右孩子为空，继续执行2；若其右孩子不为空，将右子树转执行1
void inOrder2(BiTree T)
{
    InitStack(S);
    BiTree p=T;
    while(p||!isEmpty(S))
    {
        if(p)
        {
            Push(S,p);
            p=p->lchild;
        }else{
            Pop(S,p);
            visit(p);
            p=p->rchild;
        }
    }

}

后序遍历

如果二叉树为空，则什么都不做；否则：

  1. 遍历左子树
  2. 遍历右子树
  3. 访问根节点
void postOrder(BiTree T)
{
    if(T!=NULL)
    {
        postOrder(T->lchild);
        postOrder(T->rchild);
        visit(T);
    }
}

非递归的实现

  1. 从根节点开始，将其入栈
  2. 沿其左子树一直往下收索，知道搜索到没有左孩子的结点
  3. 此时，不能出栈访问，因为如果其有右孩子，还需按相同规则对其右子树进行处理。
  4. 直到以上步骤进行不下去，若栈顶元素想要出栈被访问，要么右子树为空，要么右子树刚好被访问完（此时左子树已经被访问完了），这样就确保了正确的访问顺序。
void postOrder2(BiTree T)
{
    InitStack(S);
    BiTree p=T;
    BiTree r=NULL;//辅助指针
    while(p||!isEmpty(S))
    {
        if(p)
        {
            push(S,p);
            p=p->lchild;
        }else{
            GetTop(S,p);
            if(p->rchild&&p->rchild!=r)//如果右子树存在且没有被访问过
            {
                 p=p->rchild;
                push(s,p);
                p=p->lchild;

            }else{//否则出栈并访问结点
                pop(s,p);
                visit(p->data);
                r=p;//记录最近访问过的结点
                p=NULL;//访问完后，重置p指针

            }
        }
    }

}

层序遍历

void LevelOrder(BiTree T)
{
    InitQueue(Q);
    BiTree p;
    EnQueue(Q,T);//根节点入队
    while(!isEmpty(Q))
    {
        DeQueue(Q,p);//队头元素出队
        visit(p);
        if(p->lchild)
        {
            EnQueue(Q,p->lchild);
        }
        if(p->rchild)
        {
            EnQueue(Q,p->rchild);
        }
    }
}

线索二叉树

利用结点的空链域来指向当前结点的直接前驱和直接后继。

//线索二叉树的存储结构的定义

typedef struct ThreadNode{
    ElemType data;
    struct ThreadNode *lchild,*rchild;
    int ltag,rtag;
}ThreadNode,*ThreadTree;

/*
ltag = 0,lchild域指示结点的左孩子
ltag = 1,lchild域指示结点的直接前驱

rtag = 0,rchild域指示结点的右孩子
rtag = 1,rchild域指示结点的直接后继
*/

中序遍历线索二叉树

void InThread(ThreadTree &p,ThreadTree &pre)
{
    if(p)
    {
        InThread(p->lchild,pre);
        if(p->lchild==NULL)
        {
            p->lchild=pre;
            p->ltag=1;
        }
        if(pre&&!pre->rchild)
        {
            pre->rchild=p;
            pre->rtag=1;
        }
        pre=p;
        InThread(p->rchild,pre);
    }
}

void CreateInThread(ThreadTree T)
{
    ThreadTree pre=NULL;
    if(T)
    {
        InThread(T,pre);
        pre->rchild=NULL;
        pre->rtag=1;
    }
}

中序线索二叉树的遍历

在中序线索二叉树中招结点后继的规律：

若其标志为“1”则右链为线索，指示其后继，否则遍历右子树中第一个访问的结点（右子树最左下的结点）为其后继结点。

  1. 求中序线索二叉树中中序序列的第一个结点
ThreadNode* FirstNode(ThreadNode *p)
{
    while(p->ltag==0) p=p->lchild;
    return p;
}
  1. 求中序线索二叉树中结点p在中序序列下的后继
ThreedNode* NextNode(ThreadNode *p)
{
    if(p->rtag=0) return FirstNode(p->lchild);
    else return p->rchild
}
  1. 利用上面两个算法，可以写出中序线索二叉树的遍历序列
void InOrder(ThreadNode *T)
{
    for(ThreadNode *p=FirstNode(T;p;p=NextNode(p)))
        visit(p);
}

先序线索二叉树的遍历

void PreThread(ThreadNode &p,ThreadNode &pre)
{
    if(p)
    {
        if(p->lchild==NULL)
        {
            p->lchild=pre;
            p->ltag=1;
        }
        if(pre&&pre->rchild==NULL)
        {
            pre->rchild=p;
            pre->rtag=1;
        }
        pre=p;
        PreThread(p->lchild,pre);
        PreThread(p->rchild,pre);

    }
}

void CreatePreThread(ThreadTree &T)
{
    ThreadTree pre=NULL;
    if(T)
    {
        PreThread(T,pre);
        pre->rchild=NULL;
        pre->rtag=1;

    }
}


/*
在先序线索二叉树中找结点的后继：
1. 如果有左孩子，则左孩子就是其后继
2. 如果无左孩子但又右孩子，则右孩子为其后继
3. 如果为叶子结点，则右链域直接指示其后继
*/
ThreadNode* FindNextNode(ThreadNode *p)
{
    if(p->lchild->ltag==0)
    {
        return p->lchild;
    }
    if(p->lchild==NULL&&p->rchild->rtag==0)
    {
        return p->rchild;
    }
    if(!p->lchild&&!p->rchild)
    {
        return p->lchild;
    }
}

//先序线索二叉树的遍历
void PreOrder(ThreadTreee T)
{
    for(ThreadNode* p=T;p;p=FindNextNode(p))
        visit(p);
}

后序线索二叉树的遍历

void PriorThread(ThreadNode &p,ThreadNode &pre)
{
    if(p)
    {
        PriorThread(p->lchild,pre);
        PriorThread(p->rchild,pre);
        if(p->lchild==NULL)
        {
            p->lchild=pre;
            p->ltag=1;
        }
        if(pre->rchild==NULL)
        {
            pre->rchild=p;
            pre->rtag=1;
        }
        pre=p;
    }
}

树与二叉树的运用

排序二叉树

对二叉树进行中序遍历，可以得到一个递增有序的序列。

排序二叉树的非递归查找算法

BSTNode *BST_Search(BiTree T,ElemType key)
{
    while(T!=NULL&&key!=T->data)
    {
        if(key<T->data)
            T=T->lchild;
        else
            T=T->rchild;
    }
    return T;
}

BSTNode *BST_Search2(BiTree T,ElemType key)
{
    if(key<T->data)
        BST_Search2(T->lchild);
    if(key>T->data)
        BST_Search2(T->rchild);
    if(key==T->data)
        return T;
}

排序二叉树的插入算法

int BST_Insert(BiTree &T,ElemType k)
{
    if(T==NULL)
    {
        T=(BSTNode*)malloc(sizeof(BSTNode));
        T->key=k;
        T->lchild=T->rchild=NULL;
    }else{
        if(key==T->key)
            return 0;
        else if(k<T->key)
            return BST_Insert(T->lchild,k);
        else 
            return BST_Insert(T->rchild,k);
    }
}

二叉排序树的构造

void CreateBST(BiTree &T,KeyType key[],int n)
{
    T=NULL;
    for(int i=0;i<n;i++)
        BST_Insert(T,key[i]);
}

二叉排序树的删除

如下三种情况：

  1. 若删除的是叶子结点，则直接删除，不会改变二叉排序树的性质。
  2. 若节点只有一棵左子树或右子树，则让该节点的子树成为该节点的父节点的子树，代替该节点的位置
  3. 若z有左右子树，则令z的后继代替z，然后从二叉排序树中删除这个直接后继

二叉排序树的查找效率分析

若二叉树的左右子树的高度之差的绝对值不超过1，则这样的二叉排序树称为平衡二叉树。他的平均查找长度为O($log_2n$)。

若二叉排序树只有一个左（右）单支，其平均查找长度为O(n)。

ASL=层次*当前层次中节点的数目之和。

平衡二叉树

在插入和删除二叉树的节点时，要保证任意节点的左右子树高度差的绝对值不得超过1,。

节点左右子树高度差称为该节点的平衡因子（1,0，-1）

平衡二叉树 的插入

  1. LL平衡旋转(右单旋转)。所谓的LL是有一个节点插入到该平衡二叉树的左子树的左叶子节点上，从而导致根节点平衡因子增加到2。此时需要一次向右的旋转操作。

  1. RR平衡旋转（做单旋转）

  1. LR平衡旋转（先左后右）

  1. RL平衡旋转(先右后左)
