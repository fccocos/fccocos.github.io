иҝҷйҮҢдҝ®ж”№зҡ„жҳҜ themes/shoka/_config.yml еҶ…зҡ„й…ҚзҪ®еҸӮж•°гҖӮ

# з«ҷзӮ№еҲ«з§°

# йқҷжҖҒж–Үд»¶зӣ®еҪ•

й»ҳи®ӨеҖјжҳҜ / пјҢжҢҮдҪҝз”Ёжң¬ең°йқҷжҖҒж–Үд»¶
еҸҜд»Ҙдҝ®ж”№жҲҗ //cdn.jsdelivr.net/gh/жӮЁзҡ„githubз”ЁжҲ·еҗҚ/жӮЁзҡ„йЎ№зӣ®еҗҚ@latest/ иҝҷз§ҚеҪўејҸпјҢд»ҘдҪҝз”Ё jsDelivr иҝӣиЎҢеҠ йҖҹгҖӮ
PSпјҡjsDelivr е№¶дёҚжҳҜе®һж—¶жӣҙж–°пјҢйҮҚж–°з”ҹжҲҗж–Үд»¶еҗҺйңҖиҰҒиҖҗеҝғзӯүеҫ…

йқҷжҖҒж–Үд»¶жүҖеӨ„зӣ®еҪ•зҡ„е®һйҷ…зӣ®еҪ•еҗҚпјҢиҝҷдәӣдёҖиҲ¬дёҚж”№гҖӮ

# еҠ иҪҪи°·жӯҢеӯ—дҪ“
